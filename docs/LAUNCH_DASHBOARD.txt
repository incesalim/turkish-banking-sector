================================================================================
TROUBLESHOOTING
================================================================================

Problem: Port 8050 already in use
Solution: Try port 8080:
    python test_dashboard.py
    # Then edit the file and change port to 8080

Problem: Dashboard shows errors
Solution: Install missing package:
    pip install dash-bootstrap-components
    pip install dash

Problem: Want to stop dashboard
Solution: Press Ctrl+C in the terminal
